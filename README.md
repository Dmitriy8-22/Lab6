# Lab6: Banking System with Quadratic Equation Solver

## Components
- **banking**: Transaction and Account management library
- **solver_lib**: Quadratic equation solver library  
- **solver_app**: Command-line quadratic equation solver application
- **tests**: Unit tests for banking system

## Build
```bash
mkdir build && cd build
cmake ..
make
```

## Run tests
```bash
./tests/run_tests
```

## Run solver
```bash
./solver_application/solver_app
# Enter a b c coefficients
```
